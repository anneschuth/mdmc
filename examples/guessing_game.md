# Number Guessing Game

Implement a simple number guessing game:

1. Pick a hardcoded secret number (e.g. 42)
2. Print "Guess a number (1-100): "
3. Read a number from stdin (the user types a decimal number followed by Enter)
4. If the guess is too low, print "Too low!" and go back to step 2
5. If the guess is too high, print "Too high!" and go back to step 2
6. If the guess is correct, print "Correct! You win!" and exit with code 0

Handle only valid positive integer input. Ignore leading/trailing whitespace or newlines.
