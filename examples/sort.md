# Bubble Sort

Sort the following array of integers and print them one per line:

64, 34, 25, 12, 22, 11, 90, 1, 45, 78, 33, 55, 9, 100, 42

First print "Before:" followed by the unsorted numbers (one per line).
Then print a blank line.
Then print "After:" followed by the sorted numbers in ascending order (one per line).

Exit with code 0.
