# Maze Generator

Generate and print a simple 15x15 maze using a hardcoded seed-based approach.

Use the following characters:
- `#` for walls
- ` ` (space) for paths
- `S` for the start (top-left area)
- `E` for the end (bottom-right area)

The maze should have:
- A border of walls around the outside
- A clearly visible path from S to E
- Some internal walls creating corridors

Print the maze to stdout. Exit with code 0.
